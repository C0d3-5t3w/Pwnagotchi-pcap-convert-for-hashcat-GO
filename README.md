# Pwnagotchi-pcap-covert-for-hashcat-GO

Converts .pcap files in ~/handshakes into hashcat acceptable files in a ~/hashcatables folder

# How to:

cd ~

sudo wget " "

sudo go build pcapconvert.go

sudo chmod +x pcapconvert

sudo ./pcapconvert
